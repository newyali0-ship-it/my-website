# سجل التغييرات (CHANGELOG)

## [1.0.0] - 2026-02-07
### المزايا الجديدة:
- **واجهة المستخدم:** تصميم Glassmorphism كامل مع دعم الوضع الليلي.
- **نظام الدورات:** تحميل ديناميكي للدورات من ملف JSON مع فلترة فورية.
- **التوافق:** دعم كامل لمحررات الجوال (SPCK/Koder) بدون مكتبات خارجية.
- **الأداء:** استخدام Lazy Loading للصور و Inline SVG للأيقونات.
- **صفحة iframe:** إضافة صفحة `open-site.html` مع fallback في حال حظر العرض.

### الإصلاحات:
- حل مشكلة اختفاء القائمة في الجوال (z-index fix).
- ضمان عمل الموقع حتى لو فشل تحميل ملف JSON (Fallback Array).
